🔍 Cross-Site Scripting (XSS) Vulnerability Scanner

A simple Python-based security tool that detects Reflected Cross-Site Scripting (XSS) vulnerabilities by injecting test payloads into URL parameters and analyzing server responses.

This project is designed for educational and ethical security testing purposes.

🚀 Features

🛡️ Reflected XSS Detection
Automatically injects payloads into URL parameters and checks if they are reflected in the response.

🧪 Custom Payload Support
Add or modify your own XSS payload list for advanced testing.

🧑‍💻 Beginner-Friendly Code
Clean and easy-to-understand Python implementation.

🌐 Uses requests Library
Lightweight and efficient HTTP request handling.
